# Guide de démarrage — AvalancheJS (FR)

Ce guide est une traduction et un complément du guide officiel en anglais.  
Il a pour but d’aider les développeurs francophones à installer et utiliser **AvalancheJS**.

---

## Installation

Assurez-vous d’avoir **Node.js (>= 18)** et **npm** installés.

```bash
git clone https://github.com/ava-labs/avalanchejs.git
cd avalanchejs
npm install
npm run build
npm test
